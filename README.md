# investment-trust-3day-buy

篩選投信（投資信託）連續買進 3 天的台股股票。

## 運作邏輯

1. 從 [megatime 投信買超排行榜](https://pchome.megatime.com.tw/rank/sto4/ock04.html) 取得當日上榜股票清單。
2. 對每一檔股票，進入其個股頁面（如 `https://pchome.megatime.com.tw/stock/sto3/sid2353.html`），抓取三大法人歷史資料。
3. 檢查投信淨買賣：若最近連續 3 個交易日皆為「買超」（淨值 > 0），即列為符合條件。
4. 結果輸出至終端機，並自動儲存為 CSV。

## 安裝

```bash
pip install -r requirements.txt
```

## 執行

```bash
python3 scraper.py
```

## 偵錯（網頁結構異動時使用）

若程式顯示「無資料」，表示 megatime 的 HTML 結構可能有變動。請先執行偵錯腳本，將原始 HTML 存到 `debug_html/` 目錄，再依實際結構調整 `scraper.py` 中的解析邏輯：

```bash
# 儲存排行榜 + 前 3 檔股票的 HTML
python3 debug_html.py

# 儲存指定股票的 HTML（以台積電 2330 為例）
python3 debug_html.py --code 2330
```
